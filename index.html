Pixel Art Maker

Upload your image to make easy pixel art!
Configuration

Tutorial

The Filter Type allows you to configure the generation of the colors. By default this is none, but you can play around with the different affects available!
The Pixel Size is the size of each pixel in the picture. Don't set this too low, or else it gets exponentially longer to process! Generally, keep it above 10.
For best results, have your image be 400 × 400 or at least a square.
